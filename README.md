# 🎓 Student Attendance Viewer

A simple **React + Vite + Tailwind CSS** project that helps visualize student attendance and identify low-attendance students quickly.

This project was built to practice **React fundamentals** like `useState`, `useEffect`, filtering lists, conditional styling, and component structure.

---

## 🌐 Live Demo

You can view the live project here:

👉 https://student-attendace.netlify.app

---

## 🚀 Features

* Fetch student data from an API
* Display student name and attendance percentage
* Status badge showing attendance status
* Filter students by:

  * **All**
  * **Present**
  * **Absent**
* Highlight student row when clicked
* Toggle to show **students with attendance < 75%**
* Color-coded attendance status
* Sort students by attendance percentage

## 🖥 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/student-attendance.git
```

Navigate to the project:

```bash
cd student-attendance
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

## 🎨 UI Behavior

### Attendance Color Coding

| Attendance | Color |
| ---------- | ----- |
| ≥ 75%      | Green |
| < 75%      | Red   |

---

### Filters

Users can filter students by:

* **All Students**
* **Present Students**
* **Absent Students**

---

### Extra Controls

* Toggle to show **low attendance students (<75%)**
* Sort students by attendance percentage
* Click a row to **highlight the selected student**
